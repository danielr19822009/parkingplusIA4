🚗 Manual de Usuario y Técnico: ParkingPlus IA4

Este documento integra tanto la guía técnica para administradores de sistemas como la guía de operación para los usuarios finales del sistema ParkingPlus IA4.

🛠️ Parte I: Manual Técnico

Dirigido a: Administradores de Sistemas y Desarrolladores.

1. Arquitectura de Software

ParkingPlus IA4 utiliza un modelo desacoplado de Capas (N-Tier):

  • Backend: Node.js con el framework Express. Implementa un patrón MVC (Modelo-Vista-Controlador) para la API REST.
  • Frontend: Single Page Application (SPA) conceptual construida con Vanilla JavaScript, HTML5 y CSS3 nativo.
  • Base de Datos: MySQL Relacional con consultas optimizadas.

2. Stack Tecnológico

  • Entorno de Ejecución: Node.js v14+
  • Servidor Web: Express.js
  • Motor de BD: MySQL 8.0+
  • Middleware de Seguridad:
    • jsonwebtoken (JWT) para sesiones seguidas.
    • bcryptjs para el hashing de contraseñas de usuarios.
  • Exportación de Datos:
    • xlsx para archivos Excel.
    • html2pdf.js para reportes PDF en el cliente.

3. Esquema de Base de Datos (Tablas Clave)

Tabla Función Principal
users Credenciales, nombres y roles (Admin/Operador).
vehicles Persistencia de datos físicos (Placa, Marca, Color, Tipo).
celdas Inventario de espacios con estado (Disponible/Ocupada).
registros_ingreso Registro de entrada con fecha, hora y usuario responsable.
pico_placa Configuración de restricciones basada en JSON.

4. Instalación

  1. Clonar el repositorio.
  2. En backend/, ejecutar npm install.
  3. Configurar el archivo .env con las credenciales de MySQL.
  4. Importar database/schema.sql.
  5. Ejecutar npm run dev.

👤 Parte II: Manual de Usuario

Dirigido a: Recepcionistas, Operadores y Administradores de Patio.

1. Inicio de Sesión

  • Ingrese su correo electrónico y contraseña registrados.
  • Al ingresar, el sistema le asignará un Token de Sesión que expirará automáticamente tras un periodo de inactividad por seguridad.

2. Registro de Ingreso (Paso a Paso)

  1. En el Dashboard, presione el botón "Nueva Entrada" (o tecla F2).
  2. Digite la Placa del vehículo.
  3. El sistema verificará automáticamente si el vehículo tiene Pico y Placa. Si tiene restricción, el sistema bloqueará el registro.
  4. Seleccione el Tipo de Vehículo y la Celda asignada.
  5. Presione "Registrar". El vehículo aparecerá instantáneamente en la rejilla de "Vehículos en Patio".

3. Gestión de Novedades y Patio

  • Cada vehículo en el patio tiene un botón de "Novedad". Úselo para registrar golpes, rayones o incidentes detectados al momento del ingreso.
  • Estas novedades quedarán vinculadas permanentemente al historial del vehículo para evitar reclamos injustificados.

4. Registro de Salida

  1. Ubique el vehículo en el Dashboard o búsquelo por placa.
  2. Presione el botón "Salida".
  3. Confirme la acción. El sistema marcará la salida, liberará la celda automáticamente y actualizará las estadísticas globales.

5. Reportes y Exportación

  • Ingrese al módulo de "Reportes".
  • Filtre la información que necesite (ej. Historial completo).
  • Use los botones "Exportar a Excel" o "Exportar a PDF" para descargar la información lista para impresión o auditoría.

📄 Notas Finales

  • Seguridad: Nunca comparta su contraseña. Todas las acciones (ingresos/salidas) quedan registradas con su ID de usuario.
  • Soporte: En caso de error 500, verifique la conexión a internet y el estado del servidor de base de datos.
